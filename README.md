
## Install streamlit

```
python3 -m venv quiz_env
source quiz_env/bin/activate   # macOS/Linux
# OR on Windows: quiz_env\Scripts\activate
python -m pip install --upgrade pip
pip install streamlit
```


## Run the app via

```
streamlit run quiz_app.py
````
